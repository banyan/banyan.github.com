CloudFront を API 経由で作成するメモ

Dec 25th, 2013

CloudFront の distribution を API 経由で作成したいと思ったとする。

なんだかすごく簡単にできそうに思うけど結構骨が折れる。プログラム的な難しさは何もないけど、渡すべき値を理解しなければいけない。

Ruby を使っているとして、aws-sdk-ruby を使えばラップしてくれて何か簡単になるんじゃないか！という幻想を抱いてはいけない。 CloudFront のように渡すべきパラメータが多く、ネストしてたりすると aws-sdk-ruby を使うことでかえってめんどくさくなる気もする。

http://docs.aws.amazon.com/AWSRubySDK/latest/AWS/CloudFront/Client/V20131111.html#create_distribution-instance_method

例えば該当のメソッドは↑のリンクのような感じで、まともにハッシュを作っていくと発狂すると思う。

これをなるべく簡単にやる方法はないだろうか? 仮に aws-sdk-ruby を使うとしても AWS の公式のドキュメントは見る必要が必ずあると思う。

http://docs.aws.amazon.com/AmazonCloudFront/latest/APIReference/CreateDistribution.html
